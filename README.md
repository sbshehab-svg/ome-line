# OmeLine - Modern Stranger Chat

A premium, Omegle-inspired web application for connecting with strangers worldwide.

## Features
- **Video & Text Chat**: High-quality UI for both communication modes.
- **Advanced Filters**: Filter by Gender (Male/Female/Both), Age, and Interests.
- **Location Matching**: Real-time country detection for better matching.
- **Age Detector (Simulated)**: A cool "AI Scanner" animation on your camera feed.
- **Premium Aesthetics**: Dark mode, glassmorphism, and smooth animations.
- **Privacy First**: Secure, P2P-focused design.

## How to Run
1. Open `f:\Ome Line\index.html` in any modern web browser.
2. Grant camera/microphone permissions for the video chat feature.
3. Fill in your filters and click **Video Chat** or **Text Only**.

## Technology
- **HTML5**: Semantic structure.
- **CSS3**: Advanced animations and glassmorphism.
- **Vanilla JavaScript**: Core logic and WebRTC simulation.

*Note: This is a frontend prototype. For a production deployment, a signaling server (WebSockets/Socket.io) and STUN/TURN servers are required for P2P connections.*
